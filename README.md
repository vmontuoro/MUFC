# Manningham United Blues – Fixtures Hub

Static website for the club's home-ground schedule, overlap checks and match-day
duties. Published with GitHub Pages.

**Live site:** https://vmontuoro.github.io/MUFC/

## Pages
- `index.html` – the hub / homepage
- `Manningham_setup_packup_plan.html` – pitch setup & pack-up plan
- `Manningham_schedule_clashes.html` – schedule & pitch-rule clash check
- `Manningham_fixtures.html` – full-season fixtures & overlap check
- `Manningham_fixtures_and_overlaps.xlsx` – downloadable spreadsheet

## Updating the site
These pages are static. To refresh, regenerate the files, then:

```
git add -A
git commit -m "Update fixtures"
git push
```

The public can view the site but cannot edit it — only the repository owner can push changes.
